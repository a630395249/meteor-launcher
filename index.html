What is Meteor Launcher?

Meteor Launcher is a multi-platform desktop app to help Meteor developers create, manage and launch different Meteor applications in their computers locally.

Features

Run multiple Meteor applications with one click (No more terminals!!)
One Click
Manage all your Meteor projects from a single application
Multiple Projects
Set up the port where you want to run each Meteor application
Meteor Up
Sort all your Meteor projects like you want
Sorting
Customize each Meteor project with a color or an image
Customize
View the execution log even if the application crash
Logs

How to use it

Adding new projects

To add a new project you have only to click in the "Plus" button in the top right corner. Introduce the name of the project, select (or paste) the folder where is the Meteor App and set the port where you want to run this app. You can also select a color or an image to customize your project. By pressing the Add button your project is ready to be launched.

Editing projects

To edit a project you can right click in the tile of the project o click the cog button in the top left corner in the tile that appear when the mouse is over it. You can change all settings while the application is not running and all modifications will be stored automaticly. From this form you can also remove the project from the application (All files will be kept in his original folder).

Sorting projects

All the projects will be sorted in a horizontal scroll. You can change the position of any project by drag and drop the tile where you want. The icon in the top right corner of the tile allows you change the size of the tile (small, wide or large).

Running projects

When clicking in a project tile a window shows all actions and the running log. You can start or stop the project directly using the Play button, manage the project settings or open a terminal in the project folder to execute commands like "meteor list", "meteor update", "meteor reset", etc

The running log

The log allows you control the server state when running the project. If the application crash you can see here the error, the line and the file where something is not working properly. Debugging your apps its very important!
