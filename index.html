Meteor Launcher

Desktop App to easy manage and launch multiple Meteor applications in local

Download for Windows View on GitHub

What is Meteor Launcher?

Meteor Launcher is a multi platform desktop app to help Meteor developers create, manage and launch different Meteor applications in their computers locally.

Download now

Today it is only for Windows but we are working on make it for all desktop platforms

On Windows

Extract the zip file where you want and simply click on Meteor Launcher.exe to run the application.

Instrucctions

Adding new projects

To add a new project you have only to click in the "Plus" button in the top right corner. Introduce the name of the project, select (or paste) the folder where is the Meteor App and set the port where you want to run this app. You can also select a color or an image to customize your project. By pressing the Add button your project is ready to be launched.

Editing projects

To edit a project you can right click in the tile of the project o click the cog button in the top left corner in the tile that appear when the mouse is over it. You can change all settings while the application is not running and all modifications will be stored automaticly. From the editor panel you can also view the last execution log or remove the project from the application.

Sorting projects

All the projects will be sorted in a horizontal scroll. To change the position of any project you only need to drag and drop any tile where you want to put it. With the icon in the top right corner of the tile you can change the size of the tile (normal, wide or large).

Running projects

When you have the mouse over a tile you can see in the center of the tile a circle play button, just click on it to launch the Meteor App. If all is OK a rocket icon will appear in the left corner of the project tile and a link to open the application in your favorite browser. If something goes wrong the log window appears with the details.

Stopping projects

Simply click on the circle stop button in the middle of the tile. If you close the application with some project running, it will stop all running projects automatically.

View project Log

If you need to see the execution log you can right click on the tile when the application is running, then a window will show the actual state of the application. You can also click on the rocket icon or the "View log" link. When the project is stopped, you can access to the last execution log from the editor panel.
